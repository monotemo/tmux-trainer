PANE WRANGLER

tmux commands all start the same way: press ctrl+b, release, then press a command key. That two-step is the whole trick, and this game drills it into your hands.

Each level shows a target. Recreate it in the live terminal using real tmux bindings. 18 levels in three acts: panes, then windows, then whole sessions. The windows stacked behind the live one are part of your session — you will learn to move through them.

Beat par for three stars.

solved

stack complete

Eighteen levels, all wrangled. Panes: % " x z { } o space and arrows. Windows: c n p 0-9 , &. Sessions: d s. Open a real terminal and type tmux — it will feel like home.

cheat sheet

panes
ctrl+b % split left/right · ctrl+b " split top/bottom
ctrl+b ←↑↓→ move between panes · ctrl+b o next pane
ctrl+b x kill pane · ctrl+b z zoom
ctrl+b { / } swap pane · ctrl+b space cycle layouts
windows
ctrl+b c new window · ctrl+b n / p next / previous
ctrl+b 0-9 select by number · ctrl+b , rename
ctrl+b & kill window
sessions
ctrl+b d detach · ctrl+b s choose session
game
esc cancel prefix · r restart level
